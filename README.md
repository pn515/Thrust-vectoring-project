# Thrust Vectoring Project

This project is a thrust vectoring prototype using an Arduino, MPU6050 IMU, and two servo motors to control movement in real time.

## Components
- Arduino
- MPU6050 IMU
- 2 servo motors
- 3D printed gimbal structure

## Project function
The MPU6050 measures movement and orientation. The Arduino processes the data and sends control signals to the servo motors, which adjust thrust direction in the X and Y axes.
